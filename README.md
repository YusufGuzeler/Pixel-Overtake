# Pixel-Overtake

Bu proje, Python ve Pygame kütüphanesi kullanılarak geliştirilmiş, Nesne Yönelimli Programlama (OOP) prensiplerini temel alan dinamik bir yarış oyunudur.

## Özellikler
- **Dinamik Engel Sistemi:** Farklı hız ve ağırlıklara sahip araçların rastgele oluşturulması.
- **Gelişmiş Çarpışma Tespiti:** Pixel-perfect mask yöntemi ile hassas çarpışma kontrolü.
- **Ses Yönetimi:** Çok kanallı ses motoru (Sirenler, motor ve çarpışma efektleri).
- **Şerit Değiştirme Algoritması:** Yapay zeka kontrollü trafik araçları.

## Kullanılan Teknolojiler
- **Dil:** Python
- **Kütüphane:** Pygame
- **Konseptler:** OOP, Nesne Maskeleme, Oyun Durumu Yönetimi

## Nasıl Çalıştırılır?
1. Gerekli kütüphaneyi kurun: `pip install pygame`
2. Oyunu başlatın: `python main.py`
